CSS Fonts

The font-family property in CSS allows you to set the desired font family. It is encouraged to include several font names as a ‘fallback’ system by separating each value with a comma (the desired font family should be placed first, followed by the subsequent values, then the generic family). If the browser/operating system does not support the initial font, it will display the subsequent font or a similar font in the generic family.

HTML Code

<html>
<head>
<link rel=“stylesheet” href=“style.css”>
<head>
<body>
<p class=“p1” > The font used for this paragraph is Times New Roman.>
<p class=“p2” > The font used for this paragraph is Arial.>
<body>
<html>

CSS Code

.p1 {
font-family: “Times New Roman”, Times, serif;
}

Note: For font names that are more than one word such as Times New Roman, they must be placed between quotation marks.

.p2 {
font-family: Arial, Helvetica, sans-serif;
}

Output

The font used for this paragraph is Times New Roman.

The font used for this paragraph is Arial.

Other ways to style text

Besides specifying the font-family, you can use the font-style and font-weight properties to turn the text into italic and bold, respectively.

HTML Code

<html>
<head>
<link rel=“stylesheet” href=“style.css”>
</head>
<body>
<p class=“p1”>This text is italicized.</p>
<p class=“p2”>This text is bold.</p>
</body>
</html>

CSS Code

.p1 {
font-style: italic;
}

.p2 {
font-weight: bold;
}

Output

This text is italicized.

This text is bold.
