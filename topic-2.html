CSS Colors

Under the CSS data type is color. This is used to represent color names, RGB, HEX, HSL, RGBA, HSLA values. The CSS color property sets the foreground color value and current color value of an element's text, decorations, padding and border. In order to change font color, you need to choose which part of the HTML we want to change the text color; for this example, we want to change the heading (h1), and the two paragraphs (p1, p2). The next step is we will define the color property for the chosen HTML tags: in this case, we used color name (yellow), HEX value (#FF0000) and RGBA value (rgba(15, 10, 222, 1).

HTML Code

<!DOCTYPE html>
<head>
<link rel=“stylesheet” href=“style.css”>
<head>
<body>
<The color of the heading is Yellow.>
<The color of this paragraph is #FF0000.>
<The color of this paragraph is rgba(15, 10, 222, 1). >
<body>
<html>

CSS Code

.h1{ color: yellow; } p1{ color:#FF0000; } p2{ color:rgba(15, 10, 222, 1); }

Output

The color of the heading is Yellow.

The color of this paragraph is #FF0000.

The color of this paragraph is rgba(15, 10, 222, 1).

RGBA values are under RGB color values with alpha channel - which indicates the opacity of the color. Color names like black, red, blue and yellow are predefined and can easily show after running the code. If the user wants a more specific color, with opacity included, they may opt to use HEX or RGB. HEX values, or hexadecimal values, starts with a hashtag (#) sign, then the first 2 letters or numbers (e.g. #FF0000) refers to red, the following letters/numbers (e.g. #0080FF) indicate green and the last two numbers (e.g. #0000FF) refers to blue. Always remember that when it comes with HEX colors, it is specified with #rrggbb. With RGB values, it displays the light sources red, blue and green. RGB color values are equivalent to the intensity of colors from the lowest being 0 until the highest value being 255. For example, rgb(0, 255, 0) will show as neon green, as the middle number, 255, is equivalent to green, and the intensity of color is the highest, resulting in neon green. In RGB values, we need to take note of rgb(red, green, blue), substituting the color names with the chosen number values that indicates the intensity that will be used.
