CSS Selectors


What are Selectors?

The use of selector is to select and define the style that you want to style with CSS. CSS selector has different types which have their own syntax. In using selector there are two major ways you can do it. First one is when you have CSS and HTML in one docs you have to add CSS into the <head> of your webpage, but you can also have them in separate documents. If you keep them in separate documents you might have to label the HTML document into index.html and label the CSS documents into style.css. Index.html have to include the line of code referencing the CSS file.

HTML Code

<!DOCTYPE html>
<html lang="en">
<head>
<title> CSS Selector </title>
<link rel="stylesheet" href="style.css">
</head>
<body>
<h1>What are CSS selectors & How Do They Work?</h1>
</body>
</html>

Class Selectors

This type of selector selects only the HTML elements that have a specific class attribute. Using class selectors you must use dor(.) followers by the classname. The syntax for class selector is .classname{style properties}.

Ex. Let’s say I want to change all elements with class=”pastoral”to the color orange.

HTML Code

<h1> Not orange </h1>
<h1 class="pastoral">Very Orange </h1>

CSS Code

.patoral{
Color:orange
}

ID Selectors

However this type of selector only uses the id attributes of an element to select specific elements. The syntax for these type is #idname{style properties}

Ex. Let’s say I want to change the color and alignment of the element with the id “hubspot.”

HTML Code

<h1 id = "hubspot">#id selector</h1>

CSS Code

#hubspot{
Color:orange;
Text-align:right;
}

Output

Note: An id should not start with a number.
