CSS Selectors


What are Selectors?

The use of selector is to select and define the style that you want to style with CSS. CSS selector has different types which have their own syntax. In using selector there are two major ways you can do it. First one is when you have CSS and HTML in one docs you have to add CSS into the <head> of your webpage, but you can also have them in separate documents. If you keep them in separate documents you might have to label the HTML document into index.html and label the CSS documents into style.css. Index.html have to include the line of code referencing the CSS file.

HTML Code

<!DOCTYPE html>
<html lang="en">
<head>
<title> CSS Selector </title>
<link rel="stylesheet" href="style.css">
</head>
<body>
<h1>What are CSS selectors & How Do They Work?</h1>
</body>
</html>

Class Selectors

This type of selector selects only the HTML elements that have a specific class attribute. Using class selectors you must use dor(.) followers by the classname. The syntax for class selector is .classname{style properties}.

Ex. Let’s say I want to change all elements with class=”pastoral”to the color orange.

HTML Code

<h1> Not orange </h1>
<h1 class="pastoral">Very Orange </h1>

CSS Code

.patoral{
Color:orange
}

ID Selectors

However this type of selector only uses the id attributes of an element to select specific elements. The syntax for these type is #idname{style properties}

Ex. Let’s say I want to change the color and alignment of the element with the id “hubspot.”

HTML Code

<h1 id = "hubspot">#id selector</h1>

CSS Code

#hubspot{
Color:orange;
Text-align:right;
}

Output

Note: An id should not start with a number.

Universal Selector

The universal selector provided by CSS helps in choosing any elements within the HTML page. It goes with a single element and uses the asterisk (i.e., "*") symbol used for denoting the selector as a universal selector. It is usually written as an asterisk followed by a selector. The * is used for selecting all elements. This asterisk also has the capability to select all elements that are inside another element.
The universal selector becomes helpful when you wish to put a specific style in all the HTML elements within your web page. It can also be used for every single element that is within the element of your HTML page.

Syntax

*<
property : value;
>

HTML Code

<p class="warning">
<span lang="en-us"></span>in a red paragraph
</p>
<p id="maincontent" lang="en-gb">
<span class="warning">A red span</span>in a green paragraph</p>

CSS Code

* [lang^="en"] {
color: green;
}

*.warning {
color: red;
}

*#maincontent {
border: 1px solid blue;
}

.floating {
float: left;
}

/* automatically clear the next sibling after a floating element */
.floating + * {
clear: left;
}

Output

Grouping Selector

The CSS grouping selector is used to select multiple elements and style them together. This reduces the code and extra effort to declare common styles for each element. To group selectors, each selector is separated by a space.

HTML Code

<!DOCTYPE html>
<html>
<head>
<style>
h1, h2, p {
text-align: center;
color: red;
}
</style>
</head>
<body>

<h1>Hello World!</h1>
<h2>Smaller heading!</h2>
<p>This is a paragraph.</p>

</body>
</html>

CSS Code

h1 {
text-align: center;
color: red;
}

h2 {
text-align: center;
color: red;
}

p {
text-align: center;
color: red;
}

Output
