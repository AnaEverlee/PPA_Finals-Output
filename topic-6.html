CSS Box Model


It is a set of rules that help you determine how the items are displayed in your web page. This model has a box rectangle to be specific that is only generated for HTML elements. Each box has their own dimension, type, positioning, relationship to others, and external factors like viewport size. It consists of content, padding, a border, and margin. Box model is a box that wraps around every HTML. It also lets the users add borders to every element.

HTML Code

<!DOCTYPE html>
<head>
<title> Box Model </title>
<link rel="stylesheet" href="style.css">
</head>
<body>
<h1>The CSS box model is essentially a box that wraps around every HTML element.</h1>
</body>
</html>

CSS Content

is used to insert generated content into an element. It includes any type of text, images, and even other HTML elements. The content is the actual content of the element, it is the innermost part of the box model and is surrounded by the padding.

HTML Code

<!DOCTYPE html>
<html>
<head>
<link rel="stylesheet" href="style.css">
</head>
<body>
<div class="box">
<div class="content">
<h1>Box Model</h1>
<p>This box demonstrates the different parts of the box model in HTML and CSS.</p>
</div>
</div>
</body>
</html>

CSS Code

.box {
width: 500px;
height: 300px;
}
.content {
background-color: lightgrey;
padding: 20px;
}


Output

CSS Paddings

This is the space between the content and the border. It's a property that can be used to set the padding for all four sides of an element at once. The padding is the space around the content within the element. It is the area between the content and the border. The padding can be set using the padding property in CSS.

HTML Code

<!DOCTYPE html>
<html>
<head>
<link rel="stylesheet" href="style.css">
</head>
<body>
<div class="box">
<div class="padding">
<div class="content">
<h1>Box Model</h1>
<p>This box demonstrates the different parts of the box model in HTML and CSS.</p>
</div>
</div>
</div>
</body>
</html>

CSS Code

.box {
width: 500px;
height: 300px;
}
.content {
background-color: lightgrey;
padding: 20px;
}
.padding {
padding: 20px;
background-color: grey;
}


Output

CSS Border

is used to specify the appearance of the border around an element. The border itself can be a single line or a combination of lines (a border edge), and can have a different appearance on each side of the element. The border can be used to set the width, style, and color. The border is a line that surrounds the padding and content of the element. It is the outermost part of the box model, aside from the margins. The border can be set using the border property in CSS.

HTML Code

<!DOCTYPE html>
<html>
<head>
<link rel="stylesheet" href="style.css">
</head>
<body>
<div class="box">
<div class="border">
<div class="padding">
<div class="content">
<h1>Box Model</h1>
<p>This box demonstrates the different parts of the box model in HTML and CSS.</p>
</div>
</div>
</div>
</body>
</html>

CSS Code

.box {
width: 500px;
height: 300px;
}
.content {
background-color: lightgrey;
padding: 20px;
}
.padding {
padding: 20px;
background-color: grey;
}
.border {
border: 10px solid green;
}


Output

CSS Margins

This is the space outside the border. It's used to specify the size of the margins around an element. It can be set for individual sides of an element (top, right, bottom, left) or for all sides at once. Its values for the margin property can be specified using length units, percentages, or the keywords "auto" or "inherit". The margin is the space outside the border, around the element. It is the space between the border of the element and the other elements on the page. The margin can be set using the margin property in CSS.

HTML Code

<!DOCTYPE html>
<html>
<head>
<link rel="stylesheet" href="style.css">

<div class="margin">
<div class="border">

<div class="content">
<h1>Box Model</h1>
<p>This box demonstrates the different parts of the box model in HTML and CSS.</p>
</div>
</div>
</div>
</div>
</div>
</body>
</html>
