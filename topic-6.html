CSS Box Model

It is a set of rules that help you determine how the items are displayed in your web page. This model has a box rectangle to be specific that is only generated for HTML elements. Each box has their own dimension, type, positioning, relationship to others, and external factors like viewport size. It consists of content, padding, a border, and margin. Box model is a box that wraps around every HTML. It also lets the users add borders to every element.

CSS Content

This is used to insert generated content into an element. It includes any type of text, images, and even other HTML elements. The content is the actual content of the element, it is the innermost part of the box model and is surrounded by the padding.

Output

CSS Paddings

This is the space between the content and the border. It's a property that can be used to set the padding for all four sides of an element at once. The padding is the space around the content within the element. It is the area between the content and the border. The padding can be set using the padding property in CSS.

Output

CSS Border

This is used to specify the appearance of the border around an element. The border itself can be a single line or a combination of lines (a border edge), and can have a different appearance on each side of the element. The border can be used to set the width, style, and color.The border is a line that surrounds the padding and content of the element. It is the outermost part of the box model, aside from the margins. The border can be set using the border property in CSS.

Output

CSS Margin

This is the space outside the border. It's used to specify the size of the margins around an element. It can be set for individual sides of an element (top, right, bottom, left) or for all sides at once. Its values for the margin property can be specified using length units, percentages, or the keywords "auto" or "inherit". The margin is the space outside the border, around the element. It is the space between the border of the element and the other elements on the page. The margin can be set using the margin property in CSS.

Output

CSS Code

                .box {
      /* Set the width and height of the box */
      width: 500px;
      height: 300px;
  
    }

    /* Define a class for the content */
    .content {
      /* Set the background color of the content */
      background-color: lightgrey;

      /* Add some padding around the content */
      padding: 20px;
    }

    /* Define a class for the padding */
    .padding {
      /* Set the padding within the box */
      padding: 20px;
      background-color: grey;
    }

    /* Define a class for the border */
    .border {
      /* Add a border around the box */
      border: 10px solid green;
    }

    /* Define a class for the margin */
    .margin {
      /* Set the margins around the box */
      margin: 20px;

    }
