CSS Styling Images

To adjust the height and width of an image, you can use the height and width property and assign a value with px (pixels) as its unit of measurement.

HTML Code

<html>
<head>
<link rel=“stylesheet” href=“style.css”>
</head>
<body>
<img src=“cat.jpg” alt=“cat licking its paw”>
</body>
</html>

CSS Code

img {
height: 300px;
}

Note: If you set the image's height, the width will automatically be adjusted.

Output

The border-radius property

The border-radius property can be used to round the corners of an image (this also applies to other elements like div). You can pass one to four values for this property.

HTML Code

<html>
<head>
<link rel=“stylesheet” href=“style.css”>
</head>
<body>
<img class=“cat” src=“cat.png”>
<img class=“dog” src=“cat.png”>
</body>
</html>

CSS Code

.cat {
border-radius: 30px; (this will round all four corners equally)
}

.dog {
border-radius: 100px 0; (first value applies to top-left and bottom-right corners, and the second value applies to top-right and bottom-left corners)
}

Output
