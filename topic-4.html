CSS Border

CSS Border indicates what style, width, color, and other properties we should use with an element. It serves as somewhat of a decoration that helps spice up the website/HTML code once it is runned.
Border-style indicates what kind of border the user chose to display. There are 11 allowed border styles that can be used.

HTML Code

<!DOCTYPE html>
<head>
<link rel=“stylesheet” href=“style.css”>
<head>
<body>
<Dotted border>
<Dashed border>
<Solid border>
<Double border>
<Groove border>
<Ridge border>
<Inset border>
<Outset border>
<No border>
<Hidden border>
<Mixed border>
<body>
<html>

CSS Code

p1 { border-style: dotted; }
p2 { border-style: dashed; }
p3 { border-style: solid; }
p4 { border-style: double; }
p5 { border-style: groove; }
p6 { border-style: ridge; }
p7 { border-style: inset; }
p8 { border-style: outset; }
p9 { border-style: none; }
p10 { border-style: hidden; }
p11 { border-style: dotted dashed solid double; }

Dashed border

Solid border

Double border

Groove border

Ridge border

Inset border

Outset border

No border

Hidden border

Mixed border

Other ways to style text

With border-width, it specifies the thickness or width of the four borders. In case with their measurements, the user can indicate from the 3 predefined values namely: thin, medium and thick, or they can be specific in size with the use of measurements such as in px, pt, cm, em or etc. If you want to be really specific with the thickness of the border, remember that there are one-four values (top border, right border, bottom border, left border).

HTML Code

<!DOCTYPE html>
<head>
<link rel=“stylesheet” href=“style.css”>
<head>
<body>
<p class=“one”>Some text.</p>
</html>

CSS Code

p.one { border-style: solid; border-width: 35px 5px 9px 35px; }

Output

Some text.

Border-color is the property used to declare the color of the borders. You can also use the same property with border-width (top border, right border, bottom border, left border), just be specific with the color values to be used.

HTML Code

<!DOCTYPE html>
<head>
<link rel=“stylesheet” href=“style.css”>
<head>
<body>
<p class=“one”>A solid multicolor border.</p>
<p class=“two”>A solid one-color border.</p>
</html>

CSS Code

p.one { border-style: solid; border-color: red green blue yellow; }
p.two { border-style: solid; border-color: purple; }

Output

A solid multicolor border.

A solid one-color border.

For context, shorthand properties> in CSS are used to make coding more short and time-efficient. For border shorthand, it sets the value of border-width, border-style and border-color & list them with the same order.

HTML Code

<!DOCTYPE html>
<head>
<link rel=“stylesheet” href=“style.css”>
<head>
<body>
<p>We used the shorthand property for this paragraph.</p>
</html>

CSS Code

p { border: 10px dotted blue; }

Output

We used the shorthand property for this paragraph.

CSS Rounded Border is used to add roundness to the outer corner of the borders. We use the property border-radius to declare rounded borders for HTML.

HTML Code

<!DOCTYPE html>
<head>
<link rel=“stylesheet” href=“style.css”>
<head>
<body>
<p>You can see the rounded borders as you run the code.</p>
</html>

CSS Code

p { border: 2px solid red;
border-radius: 12px;
padding: 5px; }

Output

You can see the rounded borders as you run the code.
